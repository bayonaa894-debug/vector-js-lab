# Vector JS Lab

Aplicación web para trabajar con vectores en tiempo real usando HTML, CSS y JavaScript.

## Funcionalidades
- Visualización de vectores
- Suma de vectores
- Producto punto
- Cálculo de ángulo
- Distancia entre vectores
- Escalamiento

## Tecnologías
- HTML
- CSS
- JavaScript
- Canvas
